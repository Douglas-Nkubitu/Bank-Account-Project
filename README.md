# Bank-Account-Project

### Installation

```sh
git clone https://github.com/Douglas-Nkubitu/Bank-Account-Project.git
```

```sh
cd Bank-Account-Project/bankApp
```

• Activate the virtualenv

```sh
source bin/activate
```

• Install flask

```sh
pip3 install flask
```

• Run:

```sh
python3 bankapp.py
```
